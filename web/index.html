❔ About OmniDM

OmniDM is an AI Dungeon Master. Describe what your character does in plain language, and the DM narrates what happens. Dice rolls are real — the engine rolls them, the model only narrates the result — so a "d20" is never re-rolled to fit the story.

Getting started

Type /dm new to start a campaign, then set up your character with ⚔ Your character (or /dm join <name>). After that, anything you type that isn't a /dm command is your character's action.

Where your data lives

Playing on this device: your campaign (party, log, characters) is stored only in this browser, never on a server. Your model API key is stored only on this device, sent only to the LLM endpoint you configured, and is never logged, shown again, or saved into a game.

Connected to a server: the campaign lives on that server for everyone in the room — the model is whatever the person running the server configured, not a key you enter here.
